基础样式

  • Color（颜色） 预定义的颜色变量
  • Text（文本） 标题、段落、列表等元素的语义化样式
  • Icon（图标） 预定义图标、自定义图标等

元素

  • Layout（页面框架） Subview、导航栏、视图主体、逻辑区块等
  • Label（标签） 空心标签、实心标签、多种尺寸
  • Btn（按钮） 多种尺寸、多种类型、多种状态、多种排列组合
  • List（列表） 普通列表、链接列表、带箭头、带徽章、带值、带图标
  • Grid List（网格列表） 行列式布局的列表、弹性网络列表
  • Msg Box（提示框） 包含常规、警告、成功、错误等状态
  • Form（表单） 表单布局、表单控件、错误提示等

交互组件

  • Mask（遮罩层） 覆盖全页面的半透明遮罩层、JS API
  • Loading（加载提示） 提示用户操作正在进行、可附加文字提示、JS API
  • Dialog（弹框） 亦称对话框、JS API
  • Panel（浮出面板） 视口底部浮出操作面板、JS API
  • Scroll Box（区域滚动） 让定高容器的内容可滚动、JS API

开发者需知

  • 技术规范 编码规范、浏览器支持策略等
  • 接口 变量、mixin 等编码接口
