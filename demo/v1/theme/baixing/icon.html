Icon

概述

结构约定

所有图标建议采用 i.cmIcon 这样的结构。

初始样式

.cmIcon 类具有以下初始样式：

  • 显示为块级
  • 水平居中（此特性可能会取消）
  • 无浮动，无定位属性
  • 溢出隐藏
  • 内部文字自动隐藏（可在其内部包含文本，以增强可访问性）

图标尺寸

添加以下类名来指定图标的尺寸。

  • .cmX20 - 20px × 20px
  • .cmX30 - 30px × 30px
  • .cmX40 - 40px × 40px
  • .cmX50 - 50px × 50px

系统图标

以下图标仅供 CMUI 组件内部使用。

警告提示

自定义图标

CMUI 未对图标的实现做限制，选择权完全交给业务层。

以下图标方案都可与 i.cmIcon 正常整合：

  • 背景图片（以及 CSS Sprites）
  • Icon Font
  • SVG Icon

图标尺寸可自行定义，也可以采用上述 .cmX20 等尺寸类。
