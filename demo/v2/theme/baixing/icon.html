Icon

概述

结构约定

所有图标采用 i.cm-icon 这样的结构。

初始样式

.cm-icon 类具有以下初始样式：

  • 显示为块级
  • 无浮动、无定位属性
  • 溢出隐藏
  • 内部文字自动隐藏（可在其内部包含文本，以增强可访问性）

图标尺寸

添加以下类名来指定图标的尺寸。

  • .cm-x20 —— 20px 宽高的正方形
  • .cm-x30 —— 30px 宽高的正方形
  • .cm-x40 —— 40px 宽高的正方形
  • .cm-x50 —— 50px 宽高的正方形

系统图标

CMUI 内置了一些图标，仅供提示框、加载提示等组件内部使用。

自定义图标

CMUI 未对图标的实现做限制，选择权完全交给业务层。

以下图标方案都可与 i.cm-icon 正常整合：

  • 背景图片（以及 CSS Sprites）
  • Icon Font
  • SVG Icon

图标尺寸可自行定义，也可以采用上述 .cm-x20 等尺寸类。
