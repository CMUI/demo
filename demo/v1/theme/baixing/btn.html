Button

概述

目前仅可通过预定义的类名（而不是 mixin）来设置按钮样式。

所有按钮都应该添加 .cmBtn 类名，并根据需要添加其它辅助类名，以设置其尺寸、颜色等样式。

所有按钮默认显示为块级，可以按需对特定按钮指定宽度属性。

不建议在文本段落中混入按钮。

表单按钮

以上的各个类名除了可以用于 <a> 元素外，还可用于 <input> 和 <button> 这样的表单按钮元素。

表单内的按钮应该优先选择表单元素来实现。

表单按钮天生具备收缩包围的特性，在不指定宽度时，它们只呈现最小宽度。为使它们呈现出通长的块级效果，已对它们设置 width: 100%;。

按钮状态

悬停状态

鼠标悬停时可见此状态，不需要修改结构。

触屏设备不支持此状态。

按下状态

鼠标按下或手指触摸时可见此状态，不需要修改结构。

Android 2.X 不支持此状态。

按钮

选中状态

当某个按钮具备开关状态时，可以通过添加或除移 .active 类来实现效果。

请注意，这个类和 CSS 伪类 :active 不仅在含义上有区别，样式上也有差异。

.cmBtn.active

禁用状态

表单按钮（<button>、<input> 元素）具有 disabled 属性时，可见此状态。

对于由 <a> 元素实现的按钮，需要添加 .disabled 类。（注意：此时链接仍可响应点击动作。）

上述两种按钮禁用状态的结构和效果分别如下：

a.cmBtn.disabled
button.cmBtn @disabled

.cmBtn.disabled

.cmBtn.active.disabled
