Grid List

网格列表元素必须具有 .cmGridList 这个基础类名。

多列

可以通过指定列表项的宽度来实现对列数的控制。比如下面的 CSS 代码将列数设置为（最多）4 列，并确保每列最多可显示 6 个汉字。

#my-grid-list > li {
    width: 25%;
    min-width: 6em;
}
